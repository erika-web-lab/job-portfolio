# Portfolio Site

自身のWeb制作の実績をまとめたポートフォリオサイトです。

## 制作物一覧
- **リフォームサイト**
  - 使用技術: HTML, CSS
- **台湾観光サイト** (チーム制作: 4人×54時間)
  - 使用技術: HTML, CSS, JavaScript
- **C.C.Donuts**
  - 使用技術: HTML, CSS, PHP


## こだわった点
- **保守性**: PHP（`require_once`）を用いて、ヘッダーや共通ロジックを別ファイルで管理しています。
- **コーディング規則**: class名やID名はキャメルケースで統一し、読みやすいコードを意識しました。
- **機能性**: 掲示板機能やお問い合わせフォームの実装など、バックエンドの学習成果も反映させています。
